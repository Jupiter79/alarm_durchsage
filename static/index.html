FEHLER: FFmpeg nicht installiert!

Für die Audio-Verarbeitung und Lautstärkeanhebung wird die Software FFmpeg zwingend benötigt. Aktuell ist diese nicht auf deinem System installiert.

So behebst du das Problem unter Windows: Öffne die Eingabeaufforderung (cmd) oder PowerShell als Administrator und führe folgenden Befehl aus:

winget install "FFmpeg (Essentials Build)"

So behebst du das Problem unter Linux (z.B. Raspberry Pi): Öffne das Terminal und führe folgenden Befehl aus:

sudo apt-get update && sudo apt-get install ffmpeg -y

Starte das Programm (und ggf. den Computer) danach neu.

Internetverbindung

Hier kannst du auswählen, wie das System mit dem Internet verbunden ist.

Vorteile LAN
  • Extrem stabil und ausfallsicher
  • Kein Passwort notwendig (Plug & Play)
  • Keine Störungen durch dicke Wände oder andere Funknetzwerke
Vor-/Nachteile WLAN
  • Flexibel platzierbar (kein Kabel nötig)
  • Kann durch Wände/Störquellen schwanken
  • Achtung: Bei Router-Wechsel muss das neues Passwort eingegeben werden
Wichtige Warnung beim WLAN-Wechsel

Wenn du zu WLAN wechselst und die eingegebenen Daten (Passwort/Netzwerkname) falsch oder ungültig sind, kann sich das System nicht mehr verbinden. In diesem Notfall musst du das Gerät wieder mit einem LAN-Kabel anstecken, um wieder auf diese Oberfläche zugreifen zu können!
